Level 1：建立服務習慣

累積 150H 評核 ｜ 未過每 100H 覆核

$200/hr

職能標準對照

  • 基本禮儀：問候、敬語、微笑、儀態整潔 5: 全程主動 1: 冷淡無問候
  • 外場流程：依 SOP 執行倒茶、收桌、Set桌 5: 俐落主動 1: 多次遺漏
  • 出勤信任：準時到班、準備就緒、無臨請遲到 5: 全期準時 1: 多次遲到
  • 團隊支援：忙碌時主動收桌送餐、立即行動 5: 反應迅速 1: 拒絕消極
  • 清潔整理：確實完成區域清潔、主管抽查合格 5: 離桌即收 1: 未達標
晉升條件

● 平均評分 ≥ 4.0

● 無營運異常紀錄

● 外籍生中文 A2 通過

Level 2：穩定支援與協調

累積 300H 評核 ｜ 未過每 150H 覆核

月標準時數：75 小時
$220/hr

職能標準對照

  • 流程掌握：高峰期調整收送順序、確保流暢 5: 自主調整 1: 動線混亂
  • 顧客服務：預判顧客需求、續湯、主動道謝 5: 持續主動 1: 提醒才動
  • 餐點追蹤：監控出餐逾 20 分鐘即時回報主管 5: 主動監控 1: 未追蹤
  • 團隊合作：即時支援他區、代送收桌、主動補位 5: 主動補位 1: 不支援
  • 溝通態度：回報異常清晰、表達語氣冷靜穩定 5: 清楚冷靜 1: 語氣欠佳
晉升條件

● 平均評分 ≥ 4.25

● 展現帶領新人行為

● 外籍生中文 B2 通過

Level 3：現場主導者

累積 600H 評核 ｜ 未過每 300H 覆核

月標準時數：100 小時
$250/hr

職能標準對照

  • 現場資訊：觀察內場出單情況、主動管理顧客預期 5: 即時傳遞 1: 資訊錯誤
  • 團隊協調：明確分配任務、穩定現場節奏 5: 現場順暢 1: 分工不清
  • 溝通態度：突發狀況冷靜安撫、同仁溝通穩定 5: 安撫有效 1: 情緒急躁
  • 櫃檯應對：獨立完成結帳、外帶與電話訂位 5: 正確流暢 1: 多次錯誤
  • 責任感：無臨缺、主動承擔他人缺席之遞補支援 5: 穩定可靠 1: 拒絕配合
維持條件

● 平均 < 3.75 進入觀察期

● 一個月未改善 → 降級 L2

● 發生重大事件 → 立即觀察

營運異常

指臨請、遲到或失誤致調班或影響客情。

※ 提前調班或具證明之突發事故不在此限。

嚴重事件

致營運中斷、客訴或同仁重大壓力。

⚠️ 事件成立將立即啟動降級評估。
